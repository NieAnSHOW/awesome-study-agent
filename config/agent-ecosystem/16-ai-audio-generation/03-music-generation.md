# AI 音乐生成

> **学习目标**：能用 Suno 等工具从零生成一首完整的歌曲，掌握风格控制的技巧
>
> **预计时间**：50 分钟
>
> **难度**：⭐⭐

---

## 先说结论

2024 年之前，AI 音乐还是"听着像手机铃声"的水平。2024 年 Suno v3 发布后，AI 生成的歌曲开始出现在 TikTok 和抖音的热门 BGM 里。

**不是 AI 替代音乐人，而是 AI 让不会乐器、不懂乐理的人也能创作音乐。** 你不需要会弹钢琴，不需要懂和弦进行，只需要能描述你想要什么样的音乐。

核心工具只有一个需要记住：**Suno**。其他工具各有特色，但 Suno 是目前综合实力最强的 AI 音乐生成平台。

---

## 工具一：Suno（主力推荐）

Suno 是 AI 音乐生成领域的绝对头部。2024 年用户量突破 **1200 万**，2025 年估值约 **50 亿美元**。它能从一段文字描述生成带人声的完整歌曲。

### 注册

1. 打开 [suno.com](https://suno.com)
2. 点击 "Sign Up"，用 Google 或 Discord 账号登录
3. 免费账号立即生效

### 免费额度

| 项目 | 免费额度 |
|------|----------|
| 积分/天 | 50 积分（每天重置） |
| 生成成本 | 每首歌曲约 5-10 积分 |
| 每日可生成 | 约 5-10 首 |
| 商业使用 | 免费版不可商用，Pro 版（$10/月）可商用 |

::: tip 免费额度够用吗？
每天 50 积分，大概能生成 5-10 首歌。对于学习和尝试完全够了。如果需要持续产出商业音乐，Pro 版 $10/月，给 2500 积分，性价比很高。
:::

### 使用步骤

**Step 1：进入创作页面**

登录后点击左侧的 "Create" 进入创作界面。

**Step 2：选择模式**

Suno 提供两种创作模式：

| 模式 | 说明 | 适合 |
|------|------|------|
| **Simple（简单模式）** | 输入一段描述，AI 自动决定风格和歌词 | 新手、快速出歌 |
| **Custom（自定义模式）** | 分别指定歌词、风格、标题 | 有明确想法时 |

**Step 3：写提示词**

这是最关键的一步。提示词的质量直接决定歌曲质量。

**简单模式提示词示例：**

```
A chill lo-fi hip hop song about late night coding, with soft piano and gentle beats
```

```
一首轻快的中文流行歌，关于周末去咖啡馆的惬意心情，木吉他伴奏
```

**自定义模式提示词拆解：**

```
风格（Style of Music）：
lo-fi hip hop, chill beats, soft piano, vinyl crackle, 85 BPM

歌词（Lyrics）：
[Verse 1]
深夜的屏幕 还亮着光
代码在跳动 像心跳一样
[Chorus]
debug debug 找不到的 bug
咖啡已经凉了 但我还在想
```

::: info 提示词技巧
Suno 支持用标签来控制歌曲结构：
- `[Verse]` — 主歌
- `[Chorus]` — 副歌
- `[Bridge]` — 桥段
- `[Outro]` — 尾奏
- `[Instrumental]` — 纯音乐段落

这些标签不一定每次都生效，但用了比不用好。
:::

**Step 4：生成**

点击 "Create"，等待 30-60 秒。Suno 每次会生成两个版本，选一个你喜欢的。

**Step 5：下载和分享**

点击歌曲右下角的下载按钮，导出 MP3 或 WAV。也可以直接分享链接。

### 风格控制技巧

风格描述是 Suno 提示词的核心。以下是一些实用组合：

| 想要的风格 | 推荐描述 |
|-----------|----------|
| 短视频 BGM | `upbeat pop, catchy melody, 120 BPM, short intro` |
| 播客背景音乐 | `ambient, soft electronic, minimal, no vocals, 80 BPM` |
| 广告配乐 | `energetic, corporate, uplifting, piano and strings` |
| 游戏音效 | `retro chiptune, 8-bit, energetic, arcade style` |
| 氛围音乐 | `ambient pad, reverb, ethereal, cinematic, slow build` |
| 电子舞曲 | `EDM, house, 128 BPM, drop, synth lead, four on the floor` |
| 中文流行 | `C-pop, gentle ballad, acoustic guitar, female vocal` |

::: tip BPM 是什么？
BPM = Beats Per Minute，每分钟节拍数。60 BPM 是慢歌，120 BPM 是快歌，170+ BPM 是电子乐/说唱。指定 BPM 能帮你精确控制歌曲节奏。
:::

### Suno 的优势

- **带人声**：这是 Suno 最大的杀手锏，生成的不只是伴奏，是完整的歌
- **多语言**：支持中文、英文、日文、韩文等歌词
- **操作简单**：简单模式只需要一句话描述
- **迭代快**：不满意立刻重新生成，一次出两个版本

### Suno 的不足

- 歌曲长度限制约 4 分钟（可 extend 拼接更长）
- 歌词质量不稳定，复杂叙事容易"跑偏"
- 古典乐、爵士等复杂风格效果一般
- 生成歌曲的版权归属有争议（详见下节）

Boomy 定位是"最快上手"的 AI 音乐工具。不需要写提示词，选个风格就行。

### 注册和使用

1. 打开 [boomy.com](https://boomy.com)
2. 注册账号
3. 选择风格 → 点击生成 → 完成

### 特点

| 特点 | 说明 |
|------|------|
| **极简操作** | 选风格，一键生成 |
| **发布赚钱** | 可以直接发布到 Spotify、Apple Music |
| **风格偏电子** | 擅长 Lo-fi、EDM、Ambient 等电子风格 |
| **免费额度** | 可以免费生成和试听，下载和发布需要付费 |

### 适合场景

- 想快速出一段 BGM
- 电子音乐爱好者
- 想尝试"AI 发歌到流媒体"这件事

::: info Boomy vs Suno
Boomy 更像"音乐版的美图秀秀"——操作简单，但控制力有限。Suno 更像"音乐版的 Midjourney"——需要学习提示词，但上限更高。
:::

Mubert 专注于生成氛围音乐（Ambient Music）。不做歌曲，只做"背景氛围"。

| 时长限制 | 最长 25 分钟/首 |

## 风格控制进阶技巧

不管用什么工具，音乐风格的控制主要靠三个维度：

### 1. 流派（Genre）

```
pop, rock, jazz, classical, electronic, hip-hop, R&B, country, folk, metal, blues, reggae
```

可以组合使用：`indie folk pop`, `electronic jazz fusion`

### 2. 情绪（Mood）

```
happy, sad, energetic, calm, dark, uplifting, melancholic, nostalgic, epic
```

情绪是风格描述中最影响听感的参数。一首歌是"好听"还是"难听"，很多时候取决于情绪对不对。

### 3. 配器（Instrumentation）

```
piano, acoustic guitar, electric guitar, drums, bass, violin, synth, saxophone, trumpet
```

指定乐器能让生成的音乐更符合预期。不指定的话，AI 会自己选。

### 组合示例

```
# 安静的咖啡馆氛围
"soft jazz, warm piano, gentle bass, brush drums, 90 BPM, cozy mood"

# 激烈的游戏 Boss 战
"epic orchestral, heavy drums, brass section, intense, dark, 140 BPM"

# 播客间奏
"lo-fi hip hop, vinyl crackle, mellow, no vocals, 80 BPM, chill"
```

## 常见问题

### AI 生成的音乐有版权吗？

这个问题目前没有全球统一的答案。简单说：

| 情况 | 版权状态 |
|------|----------|
| 你付费使用 Suno Pro 生成 | Suno 授权你商用，但版权归属有争议 |
| 你用免费版生成 | 通常不可商用 |
| 你用 AI 辅助创作（写了歌词，AI 编曲） | 版权更有可能属于你 |
| 纯 AI 自动生成 | 多数国家不授予版权 |

::: warning 重要提醒
如果你的音频内容要公开发布或商业使用，务必阅读平台的用户协议。AI 音乐的版权法律仍在快速演变中。
:::

### AI 音乐能替代真人音乐人吗？

短期内不能。原因：

1. **情感表达**：AI 缺乏真正的情感理解，音乐的情绪是"模拟"出来的
2. **即兴创作**：现场演出的即兴发挥，AI 做不到
3. **复杂编排**：交响乐、歌剧等复杂编排，AI 的质量还差得远

但 AI 音乐在"功能性音乐"（BGM、广告配乐、游戏音效）这个领域，已经能替代大量传统工作。
