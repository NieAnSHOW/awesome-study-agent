### 2. 新增国产 IDE 独立章节

#### Trae（字节跳动）

**定位**：国内首个 AI 原生 IDE，从底层架构将 AI 融入开发流程。

**国内版 vs 国际版对比**：

| 维度 | 国内版 | 国际版 |
|------|--------|--------|
| 定价 | 完全免费（个人） | Pro $20/月，用量计费 |
| 模型 | 豆包 1.5 Pro、DeepSeek R1/V3、通义千问 | Claude Sonnet、GPT-4o、Gemini |
| 安装包 | ~214MB（国内优化） | ~214MB |
| 网络 | 国内直连，网络稳定 | 需国际网络 |
| 生态 | 字节云生态集成 | 云平台无关 |
| 特色 | Builder 模式，中文理解一流 | Builder 模式，模型更强 |

**定价**：
- 国内版：个人完全免费
- 国际版：Pro $20/月（含一定量高速请求），超额按量计费

**功能亮点**：
- Builder 模式：意图→结构→实现的 AI 原生工作流
- SOLO 智能体闭环
- Chat + Composer 双模式
- MCP 生态支持

**适用场景**：
- 国内开发者首选
- 预算有限想体验 AI 原生 IDE
- 需要稳定网络访问

#### CodeBuddy（腾讯云）

**定位**：腾讯云出品的轻量化 AI 代码助手，主打多端协同和腾讯生态集成。

**核心能力**：
- 支持 200+ 编程语言
- 基于混元大模型（Hunyuan），专为腾讯生态定制
- 企业级安全扫描和私有化部署
- VS Code / JetBrains 插件形态
- 深度集成 Cloud Studio

**定价**：
- 个人版：可申请免费使用
- 企业旗舰版：¥198/人/月（2026年5月调价后，原价¥78）
- 企业专享版（专有云）：¥316/人/月

**适用场景**：
- 腾讯云 / 微信小程序 / 企业微信生态开发者
- 国内企业团队
- 需要私有化部署的组织
