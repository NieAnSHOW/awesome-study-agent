# WorkBuddy 实践

> **学习目标**: 理解腾讯 WorkBuddy 的产品定位和核心能力 — 掌握 12 个真实办公场景的 WorkBuddy 实战用法 — 学会安装配置、远程操控和安全使用
>
> **预计时间**: 1.5-2.5 小时
>
> **难度等级**: ⭐⭐☆☆☆

::: tip 前置知识
建议具备 **AI Agent 基本概念**，了解智能体是什么以及能做什么即可。
:::

---

## 模块介绍

WorkBuddy 是**腾讯云 CodeBuddy 团队**于 2026 年 3 月 9 日正式推出的 AI 原生桌面智能体工作台。它是一款真正能"听懂人话、带脑子思考、实际操作本地文件"的桌面智能体——用户只需用一句话描述需求，WorkBuddy 就能自主规划执行步骤，调用内置能力，交付可直接使用的多模态成果（文档、表格、PPT、海报等）。

WorkBuddy 基于 **CodeBuddy 自研智能体架构**（腾讯内部覆盖 90%+ 工程师），兼容 **OpenClaw 技能生态**和 **MCP 协议**，内置混元、DeepSeek、GLM、Kimi、MiniMax 等多模型支持，支持企业微信、QQ、飞书、钉钉远程操控。

本文收录 **12 个真实应用场景**，按通用办公、商务与市场、技术研发、创意与设计四大方向组织。

本模块是**实践手册**而非理论教程。六个章节以"理解定位 → 行业案例 → 用前必读"为主线，让你在动手操作中快速上手。

## 学习检验

完成本模块后，检查你是否掌握以下要点：

- [ ] 理解 WorkBuddy 的产品定位和核心能力
- [ ] 能根据自身工作场景选择合适的案例参考
- [ ] 能安装配置 WorkBuddy 客户端、注册并熟悉界面
- [ ] 能设置远程操控（企业微信、飞书等）和定时任务
- [ ] 理解积分（Credit）体系，知道免费额度与获取方式
- [ ] 能识别使用 AI 桌面智能体时的安全考量
