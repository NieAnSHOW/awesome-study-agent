## 二、AI 绘画概述 (01-overview.md)

### 1. 什么是 AI 绘画？

- 定义：通过 AI 技术从文本/图像输入自动生成视觉内容
- 三大技术路径简介：扩散模型（Diffusion）、GAN、自回归模型
- 与人类创作的核心区别

### 2. 技术发展历程

| 阶段 | 时间 | 标志事件 | 能力 |
|------|------|---------|------|
| 萌芽期 | 2015-2020 | DeepDream、StyleGAN、VQ-VAE | 风格化滤镜、人脸生成 |
| 爆发期 | 2021-2022 | CLIP、DALL-E 2、Midjourney v3、SD 1.4 | 文生图进入大众视野 |
| 成熟期 | 2023-2024 | SDXL、Midjourney v6、DALL-E 3、Firefly | 商用级质量、可控性提升 |
| 普及期 | 2025-2026 | SD3/Flux、Midjourney v7、实时生成 | 高质量自制、版权安全、行业融合 |

### 3. 行业格局

- 2026 年市场整体规模数据
- 主流平台格局：Midjourney（风格）、DALL-E 3（文字/语义）、SD/Flux（开源/自控）、Firefly（版权/集成）
- 中国市场：通义万相、文心一格、即梦 等格局
- 用户画像：设计师、营销人、电商、自媒体等

### 4. 能力边界

- **能做**：写实人像/风景、概念设计、产品渲染、创意插画、风格迁移、局部重绘、图像扩展
- **暂不能**：精确透视图、长文本排版、多人复杂交互、动态变化（需视频生成）

### 5. 与 AI 视频生成的对比

- 技术路径异同（扩散模型 vs 时空扩散）
- 工具与生态关系
- 综合创作流程：生图 → 局部修改 → 图生视频

### 难度：⭐⭐ | 篇幅：~300 行

## 三、主流平台详解 (02-main-platforms.md)

### 1. Midjourney

- **定位**：风格化最强、艺术性最高的 AI 绘画平台
- **核心能力**：v6.1/v7 版本的新特性（真实感提升、文字渲染改进、角色一致性）
- **费用**：$10/月（基础）-$60/月（Pro），年付折扣
- **操作方式**：Discord 命令交互、Web 版新界面（2025+）
- **参数体系**：`--ar`、`--s`、`--c`、`--v`、`--stylize`、`--iw` 等
- **特色功能**：Blend、Describe（图生文）、Pan/Reframe、角色参考
- **优势**：美学调校最佳、社区活跃、持续迭代
- **劣势**：非实时交互、无局部重绘、隐私受限
- **适用场景**：创意概念、广告视觉、艺术设计

### 2. DALL-E 3

- **定位**：OpenAI 的文生图引擎，提示词理解最精准
- **集成方式**：ChatGPT Plus/Pro 内置 + API
- **核心能力**：文字渲染、复杂指令遵循、精准语义理解
- **限制**：内容安全策略较严、风格可控性不如 MJ/SD
- **适用场景**：快速原型、插画、教育内容、概念验证

### 3. Adobe Firefly

- **定位**：版权安全的商业级图像生成工具
- **集成生态**：Photoshop、Illustrator、Express 深度集成
- **核心技术**：Firefly Image 3 模型、生成填充（Generative Fill）
- **版权优势**：训练数据来源经过审核，适合商业使用
- **适用场景**：商业设计、品牌物料、版权敏感项目

### 4. Stable Diffusion 3 与 Flux

- **SD3**：Stability AI 最新开源模型，支持多种 MMDiT 架构
- **Flux**：Black Forest Labs 的 2025 开源新模型，质量对标 Midjourney
- **生态对比**：开源自由度 vs 商业平台的便捷性
- **适用场景**：需要自部署、定制化、批量生成的专业用户

### 5. 平台对比总表

| 维度 | Midjourney | DALL-E 3 | Firefly | SD/Flux |
|------|-----------|---------|---------|---------|
| 图像质量 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| 提示词遵循 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| 可定制性 | ⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 商业版权 | 需付费 | ChatGPT 版受限 | ✅ 友好 | 自部署自由 |
| 价格区间 | $10-60/月 | $20/月（含ChatGPT） | $5-55/月 | 免费（自托管）|

## 四、开源生态与本地部署 (03-open-source.md)

### 1. Stable Diffusion 生态谱系

- **模型代际**：SD 1.4/1.5 → SD 2.1 → SDXL → SD3 → SD3.5
- **Flux 新势力**：Flux Pro/Dev/Schnell 三档
- **Checkpoint**：主流社区模型（Realistic Vision、DreamShaper、Juggernaut 等）
- **模型获取**：Civitai（社区模型库）、Hugging Face（官方模型）

### 2. 部署方案

| 方案 | 特点 | 推荐人群 |
|-----|------|---------|
| **Auto1111 WebUI** | 最成熟、插件最多、教程丰富 | 初学者入门的 WebUI |
| **ComfyUI** | 节点式工作流、灵活高效、2025+ 主流推荐 | 进阶用户、工作流开发者 |
| **Fooocus** | 极简界面、一键安装、Midjourney 式体验 | 快速上手用户 |
| **Forge** | SD3/Flux 优化、显存优化好 | 硬件有限用户 |
| **云端方案** | RunPod、Modal、Replicate 等 | 无 GPU 用户 |

- 各方案的安装要点和环境要求

### 3. 核心控制技术

- **ControlNet**：
  - Canny（边缘检测）、Depth（深度图）、OpenPose（姿态）、Scribble（涂鸦）、Lineart（线稿）
  - 组合使用技巧、多 ControlNet 联合控制

- **LoRA**：
  - 什么是 LoRA（低秩适配）
  - 角色 LoRA、风格 LoRA、概念 LoRA
  - LoRA 训练入门（Dreambooth vs LoRA vs 全量微调）
  - 权重控制与叠加技巧

- **IP-Adapter**：图像提示适配器，图生图的图像条件控制
- **T2I-Adapter**：类似 ControlNet 的轻量适配方案
- **InstantID/Pulid**：人脸保持与角色一致性方案

### 4. 图像编辑与后处理

- **Inpainting**：局部重绘技术（遮罩 + 重绘）
- **Outpainting**：图像扩展
- **Upscaling**：分辨率提升（ESRGAN、Ultimate SD Upscale）
- **Batch 处理**：批量生成与自动化工作流

### 难度：⭐⭐⭐ | 篇幅：~450 行

## 六、提示词工程与创作 (05-prompt-and-creation.md)

### 1. 提示词基础结构

**四要素框架**：
- **主体**（Subject）：人物/物体/场景描述 
- **环境**（Environment）：背景、光照、氛围
- **风格**（Style）：艺术风格、媒介、流派
- **参数**（Parameters）：平台特定的技术参数

**公式化表达**：
```
[主体描述] + [动作/状态] + [环境背景] + [光照氛围] + [风格媒介] + [质量修饰词]
```

### 2. Midjourney 提示词体系

- 基础提示词结构
- 参数详解：`--ar`（宽高比）、`--s`（风格化）、`--c`（混乱度）、`--iw`（图像权重）、`--v`（版本）、`--no`（排除）
- 多提示词（`::` 分割权重）
- Remix 模式
- Permutation 批量生成
- 提示词示例库（写实、动漫、概念艺术、产品、建筑等）

### 3. Stable Diffusion 提示词体系

- 正面提示词 vs 负面提示词
- 权重语法：`(word:1.2)`, `[word:0.8]`, `word+`, `word-`
- Embedding（Textual Inversion）
- 提示词模板与样式

### 4. 风格控制进阶

**风格类型库示例**：
```markdown
- 摄影类：Cinematic、Photorealistic、Analog、Film Grain、Portrait Photography
- 艺术类：Oil Painting、Watercolor、Sketch、Anime、Pixel Art、Ukiyo-e
- 质量词：8K、Masterpiece、Trending on ArtStation、Award Winning
- 光照词：Golden Hour、Studio Lighting、Dramatic Lighting、Volumetric Lighting
- 构图词：Close-up、Wide Shot、Bird's Eye View、Dutch Angle
```

### 5. 工作流实战

**文生图标准流程**：
```
构思描述 → 初版生成 → 评估效果 → 迭代优化 → 精修定稿 → 后处理增强
```

**图生图（Img2Img）**：
- 源图像 + 提示词 + Denoising Strength
- 风格迁移、草图转成稿、产品变体

**局部重绘（Inpainting）**：
- 遮罩生成技术（手绘/自动）
- 修复特定区域而不影响整体

**图像扩展（Outpainting/Expansion）**：
- 向四个方向扩展画布
- AI 自动补全内容

**ComfyUI 工作流示例**（文字 + 图示）：
- 文生图基础工作流
- ControlNet 姿态控制工作流
- 批量换脸工作流

### 6. 质量评判与常见问题

**质量评估维度**：
- 语义符合度（Prompt adherence）
- 构图与光影（Composition & Lighting）
- 细节丰富度（Detail richness）
- 物理合理性（Physical plausibility）

**常见问题与修复**：
| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 手指/手部畸形 | 模型对手部建模不足 | 负提示词、手部控制 LoRA、Inpainting 修复 |
| 语义遗漏 | 提示词太长/权重不够 | 简化提示词、调整权重 |
| 风格不一致 | ControlNet 权重过高或过低 | 调整 ControlNet 权重比例 |
| 背景杂乱 | 未指定背景 | 增加环境描述、使用负提示词 |

## 七、商业应用与案例 (06-business-cases.md)

- **角色概念**：角色设计的快速迭代
