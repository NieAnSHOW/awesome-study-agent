# 模块一:AI 概述与 Agent 概念

> **学习目标**:建立对 AI 和 Agent 的基本认知,理解为什么 Agent 是下一代 AI 的核心
>
> **预计时间**:2-3 小时
>
> **前置知识**:无
>
> **最后更新**:2025 年 1 月

## 模块介绍

欢迎来到 AI Agent 的世界!这是你系统性学习之旅的第一步。

本模块会涵盖:
- **AI 是什么** - 人工智能的基本定义和核心特征
- **AI 如何发展至今** - 从图灵测试到 ChatGPT 的演进历程
- **什么是 AI Agent** - Agent 的概念和独特之处
- **为什么 Agent 重要** - Agent 作为下一代 AI 的核心价值

### 为什么从这一模块开始?

McKinsey 2025 年全球 AI 调查报告显示[^1]:
- **几乎所有的组织**都在使用 AI
- **62% 的组织**正在实验或使用 AI Agent
- 但**近三分之二**的组织仍处于早期阶段

AI Agent 的浪潮才刚刚开始,现在正是学习的好时机。

通过本模块,你会建立对 AI 和 Agent的认知框架,为后续学习打下基础。

### 学习方法建议

本模块采用渐进式学习法:
1. **概念理解** - 理解核心概念
2. **案例分析** - 通过真实案例加深理解
3. **思考检验** - 每章结尾的思考题检验学习效果
4. **实践探索** - 动手体验相关 AI 产品
