# MarkText：极简开源的 Markdown 编辑器

> 2026-09-01 即将停止服务，但目前仍可正常使用。

MarkText 是一款开源、跨平台的 Markdown 编辑器。卖点很简单——**打开就能用，所见即所得**。不像 Obsidian 要搭建知识体系，也不像 LogSeq 要学大纲笔记法。MarkText 就是让你打开 `.md` 文件直接读或写，没有多余的负担。

## 核心特点

- **三种编辑模式**：源码模式（看原始 Markdown）、打字机模式（居中打字，专注输入）、专注模式（只高亮当前段）
- **实时渲染**：不是分屏预览那种左右对照，而是一边写一边渲染出最终样式
- **极简界面**：没有侧边栏，没有插件市场，没有数据库概念——就是一个纯粹的文件编辑器
- **跨平台免费**：Windows/macOS/Linux 全支持，MIT 开源协议，GitHub 48k+ star

## 快速上手

1. 下载安装，一路下一步
2. 打开软件，看到一个干净的空白编辑区
3. 拖入一个 `.md` 文件，或直接开始打字
4. 完事

没有"新建知识库"、没有"选择 vault 路径"。MarkText 甚至不记住上次打开的文件——它就是纯粹的文件编辑器，打开即用。

## 核心功能

- **编辑模式切换**：底部状态栏一键切换三种模式
- **图片粘贴**：可直接粘贴图片，默认存为本地文件
- **导出**：支持导出为 HTML、PDF、纯文本
- **主题切换**：暗色/亮色主题自由切换
- **数学公式**：支持 LaTeX 公式渲染

## 优缺点

**优点**：零学习成本，上手最快。渲染漂亮，开箱即用。纯本地运行，不跟踪不联网。

**缺点**：已停止维护——这是最大的问题。项目于 2024 年放弃维护，2026-09-01 将完全停止服务，不会有新功能也不会有安全更新。不支持插件扩展，没有文档管理功能，不适合大型写作项目。社区已停止活跃。

## 小结

MarkText 最适合的场景：**电脑里扔着十几个 `.md` 文件，快速打开看一眼或改两笔**。它不改变你的工作流，也不逼你学新概念。但要记住它的寿命——距离服务终止只剩几个月了。作为主力编辑器风险不小，建议同时准备一个替代品，比如 Obsidian 或 VS Code 的 Markdown 插件。
