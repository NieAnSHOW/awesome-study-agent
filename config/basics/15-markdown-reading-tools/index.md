# 模块十五：Markdown 阅读工具

> **学习目标**：掌握几款免费好用的 Markdown 阅读和管理工具，让你的学习资料管理不再乱糟糟

> **预计时间**：3-5 小时（选一款上手即可）

> **前置知识**：
> - 会基本的电脑操作（下载文件、双击安装）
> - 知道什么是文件路径

## 这个模块是干嘛的

先说清楚——这个模块不教你 AI Agent。

它解决的是一个问题：你下载了一堆模型配置文件、项目 README、技术博客，打开一看全是乱码一样的 Markdown 原始格式。或者更糟——你收藏了一堆学习资料，散落在桌面、下载文件夹、微信传输助手里，需要用的时候永远找不到。

这个模块就是帮你解决这个问题的。几款免费工具，下载安装，打开即用。

## 等等，Markdown 是什么？

它不是一种编程语言，也不是某个软件的文件格式。Markdown 是一种纯文本标记语法——用 `#` 表示标题、`**` 表示加粗、`-` 表示列表。简单到你五分钟就能学会，强大到 GitHub、Notion、Obsidian 都在用它。

几乎所有 AI 项目的文档都使用 Markdown。你读到的这份教程也是。

|------|-----------|--------|

## 新手建议

如果你完全没头绪，从 **Obsidian** 开始。它的社区最大，遇到问题随便搜就有答案。如果你只是偶尔打开 .md 文件读一读，**MarkText** 打开最快。

不用纠结选哪个——这几款都免费，都支持导入导出。今天用了 Obsidian，明天想换 Logseq，数据一键就能搬过去。先做选择，再优化。**先做出来，再做到好。**

现在选一个工具，点开看看吧。不是花几个小时研究哪个最好，是花十分钟下载安装、五分钟新建第一个笔记。
