# 横向对比与选型建议

> 选工具不是选最好的，是选最合适的。

| 开源免费 | 个人免费，同步付费 | 完全免费（MIT） | 完全免费（MIT） | 完全免费 | 完全免费 |

## 怎么选

别纠结，看你的场景：

- **只读不写** → 选 **MarkText**。打开最快，装上就能用。但注意它马上停止维护了，建议用 Obsidian 做替代
- **记笔记、搭知识体系** → 选 **Obsidian**。社区最大、教程最多。新手入门第一选择
- **喜欢结构化、习惯列提纲** → 选 **Logseq**。大纲式输入一旦习惯，效率很高
- **多设备同步刚需** → 选 **Joplin**。免费同步方案最成熟，手机电脑全覆盖
- **写论文、搞学术** → 选 **Zettlr**。引用管理 + 导出功能，其他工具替代不了

## 重要提醒

这几款工具都免费，数据都基于标准 Markdown 文件。今天用 Obsidian，明天想换 Logseq——把文件夹搬过去就行。**先选一个用起来，比花一周研究"哪个最好"值一百倍。**
