# 通用办公案例

> **学习目标**: 掌握 WorkBuddy 在通用办公场景中的实际应用
> **预计时间**: 30 分钟
> **难度等级**: ⭐⭐☆☆☆

本章收录 3 个通用办公场景案例，涵盖本地文件管理、数据处理和信息提取等日常高频任务。每个案例包含完整的指令示例和交付成果说明，你可以直接参考使用或稍作调整适配到自己的工作中。

---

## 案例 1：本地文件批量整理

**一句话**: 让 WorkBuddy 自动将杂乱的下载文件夹按类型分类归档

**适用人群**: 所有职场人

**前置条件**: 本地有一个待整理的文件目录（如下载文件夹）

### 场景背景

"桌面太乱了帮我整理一下"——这大概是每个打工人电脑上都积压了的念头。下载文件夹常常混杂着各种文档、图片、压缩包、安装程序，日积月累堆积成几百个文件，找东西全靠搜索。

有了 WorkBuddy，你只需要一句指令，它就能自动扫描文件夹、识别文件类型、创建分类子目录并移动归档。整个过程在本地执行，文件不离机，安全可控。

### 指令示例

```
用户：帮我把桌面上的"下载"文件夹按类型（图片、文档、压缩包、视频）自动分类，每种类型放到对应的子文件夹中。
```

WorkBuddy 会执行以下操作：

1. 扫描指定目录，识别所有文件的扩展名和真实类型（不只看后缀，还会校验文件头）
2. 根据识别结果将文件归入对应类别——图片类（jpg/png/gif/svg/webp）、文档类（docx/pdf/xlsx/pptx/txt）、压缩包类（zip/rar/7z/tar.gz）、视频类（mp4/mov/avi/mkv）
3. 对无法归类的文件，统一放入"其他"文件夹
4. 自动避免文件名冲突（同名文件追加数字后缀）

### 交付成果

整理后的目录结构：

```
下载/
├── 图片/              ← 所有图片文件
│   ├── 2025年会合影.jpg
│   ├── 产品截图.png
│   └── logo设计稿.svg
├── 文档/              ← 所有文档文件
│   ├── Q2述职报告.docx
│   ├── 合同模板.pdf
│   └── 会议纪要.txt
├── 压缩包/            ← 所有压缩文件
│   ├── 项目源码.zip
│   └── 素材包.rar
├── 视频/              ← 所有视频文件
│   └── 培训录像.mp4
└── 其他/              ← 无法归类的文件
    ├── installer.exe
    └── config.ini
```

整理完成后，WorkBuddy 会同步生成一份整理清单（Excel），记录每个文件的原始位置、新位置和文件大小，方便日后追溯。

### 扩展思路

这个能力可以延伸覆盖更多场景：

- **HR 场景**：将收到的简历文件按岗位名称自动归类（"前端开发-张三.pdf" → 前端岗/、后端岗/）
- **项目文档归档**：项目结束后，将各版本文档、设计稿、会议记录按阶段分类归档
- **照片整理**：将手机导出的照片按拍摄日期自动建立文件夹（2026-01/、2026-02/...）

## 案例 2：Excel 数据清洗与分析

**一句话**: 对销售数据/报表进行批量清洗、汇总和可视化

**适用人群**: 运营、销售、财务等需要处理表格的人

**前置条件**: 有一个包含原始数据的 Excel 文件

### 场景背景

从系统导出的原始数据几乎从来不是"干干净净"的——可能存在空值行、重复记录、格式不统一、日期混乱等问题。手动清洗不仅耗时，而且容易遗漏。

WorkBuddy 可以直接读取 Excel 文件，自动完成数据清洗（删除空行、去重、格式化），然后按你的要求进行汇总分析，最后生成可视化图表。整个过程在本地执行，数据不会离开你的电脑。

### 指令示例

```
用户：读取桌面的"销售数据.xlsx"，清洗所有空值和重复行，按部门汇总销售额，生成柱状图保存到桌面。
```

WorkBuddy 会执行以下操作：

1. 读取 Excel 文件，识别表头和各列数据类型
2. 数据清洗：
   - 删除完全空白的行
   - 删除重复记录（基于关键字段去重）
   - 修正格式异常（如日期统一为 yyyy-mm-dd、金额转为数字格式）
   - 标记或补全缺失的非关键字段
3. 按部门汇总销售额，计算各部门的总额、平均值、最大/最小值
4. 基于汇总数据生成柱状图，直观展示各部门销售业绩对比

### 交付成果

- **清洗后的数据文件**：原始数据经过清洗后另存为新文件，原文件不受影响
- **汇总统计表**：在同一 Excel 文件的新 Sheet 中生成汇总数据
- **可视化图表**：柱状图以图片形式保存到桌面，也可嵌入 Excel 文件中

最终输出效果类似：

```
销售数据汇总（2026年3月）
━━━━━━━━━━━━━━━━━━━━━━━━
部门         销售额(元)    占比
市场部       1,285,600    31.2%
销售部       1,560,200    37.9%
产品部        685,400     16.6%
研发部        586,300     14.3%
━━━━━━━━━━━━━━━━━━━━━━━━
合计        4,117,500    100%

  - 原始行数：1,256 行
  - 删除空行：23 行
  - 删除重复：18 行
  - 清洗后行数：1,215 行
  - 数据质量：99.2%
```

柱状图可在桌面上直接查看，也可粘贴到周报或汇报 PPT 中。

### 扩展思路

- **财务报表分析**：读取月度收支表，自动计算毛利率、费用率、同比环比
- **库存数据核对**：比对系统库存和盘点数据，标记差异项生成差异报告
- **考勤数据处理**：读取打卡记录，统计迟到/早退/缺勤，计算月度出勤率

## 案例 3：票据信息提取

**一句话**: 从多张发票/票据图片中批量提取关键信息并汇总

**适用人群**: 财务、行政

**前置条件**: 文件夹中保存了多张发票图片或 PDF

### 场景背景

每到月底报销季，财务和行政同事最头疼的就是整理一堆发票。一张一张打开、手动录入金额、日期、发票号——这种机械重复的工作不仅效率低下，而且容易出错（手误输错数字、漏看某张发票）。

WorkBuddy 利用多模态识别能力，可以一次性读取文件夹中的所有发票图片或 PDF，自动识别并提取关键字段，最终汇总输出为结构化的 Excel 表格。你只需要复核一次，就能完成原本需要数小时的手工录入。

### 指令示例

```
用户：读取桌面的"发票"文件夹，提取所有发票的金额、日期和发票号，汇总输出到"发票汇总.xlsx"。
```

WorkBuddy 会执行以下操作：

1. 扫描"发票"文件夹，识别所有图片和 PDF 文件
2. 逐张读取并进行多模态识别（支持增值税专用发票、普通发票、电子发票、卷式发票等多种格式）
3. 提取每张发票的关键信息：发票号码、开票日期、金额（含税/不含税）、购买方名称、销售方名称
4. 对识别置信度低于 90% 的字段自动标记为"需人工复核"
5. 将所有提取结果汇总输出到 Excel 文件

附件中的原始发票文件会按发票号码重命名，一并归档到同级目录，方便你随时核查。

### 扩展思路

- **合同关键条款提取**：读取合同 PDF 文件夹，提取每份合同的甲方乙方、合同金额、签署日期、有效期，汇总输出
- **名片信息录入**：将收到的名片拍照汇总到一个文件夹，提取姓名、公司、职位、联系方式，导出为通讯录
- **报销单批量处理**：读取员工提交的报销单图片，提取金额和项目信息，自动填入报销台账

## 扩展思路

以上 3 个场景覆盖了通用办公中最常见的任务类型。你可以根据自身工作需求进行类推和组合：

- **HR 场景**：简历筛选材料整理、入职文档模板生成
- **财务场景**：月度报销单汇总、银行流水自动对账
- **行政场景**：资产登记表整理、会议纪要归档
