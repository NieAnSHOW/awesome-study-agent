# Joplin：开源跨平台的 Markdown 笔记应用

## Joplin 是什么

Joplin 是一款开源、跨平台的笔记应用，核心是 Markdown。不是 Obsidian 那样的知识图谱工具，也不是 Notion 那样的全能工作台——**Joplin 就是让你在不同的设备上，都能记同一套笔记**。

你的笔记写在 Markdown 里，可以加密同步到自己的云存储。手机、电脑、平板，随时打开，内容一致。

## 核心特点

- **端到端加密**：笔记在上传到云端之前就加密了。服务商看不到内容，只有你自己能解密。不是"服务商说不会看"，而是"技术上就看不了"
- **多种同步方式**：支持 NextCloud、Dropbox、OneDrive、AWS S3、Joplin Cloud，甚至 WebDAV 协议。**你的同步方式你说了算**，不是只能用它家的云
- **内置待办管理**：不只是记笔记，还能打勾。创建待办列表，打勾完成，同步勾选状态——把它当 To-do 应用用也没问题
- **Web Clipper 浏览器插件**：看到好文章，一键剪藏到 Joplin。支持 Firefox 和 Chrome。剪下来的自动转 Markdown，保留原文链接和时间戳
- **丰富插件和主题**：从表格增强到日历视图，从代码高亮到画板绘图。虽然不如 Obsidian 的插件生态庞大，但够用

## 快速上手

### 第一步：创建笔记本

打开 Joplin，左侧默认有个"笔记本"分类。右键点击 → 新建笔记本，取个名字——比如"AI 学习笔记"。笔记本就是你的顶层分类，类似文件夹的概念。

### 第二步：写第一段笔记

选中笔记本，点顶部的"新建笔记"按钮。开始写：

```
# 什么是 AI Agent

AI Agent = 大模型 + 工具调用 + 记忆。

可以理解为一个**能自己调用工具的 AI**。
```

Joplin 使用的是标准 Markdown，语法和 GitHub 上一样。标题、列表、代码块、图片，写法都一样。

### 第三步：加标签

写完笔记，在右侧属性区添加标签，比如 `#ai-agent`、`#beginner`。标签和笔记本是两套分类系统——笔记本是文件夹，标签是横切标记。同一篇笔记可以打多个标签，但只能属于一个笔记本。

## 核心功能速览

**搜索**：支持全文搜索。标题、正文、标签都能搜。快捷键 `Ctrl+P`（macOS 用 `Cmd+P`）快速跳转。

**同步**：设置 → 同步 → 选择同步目标。配置一次之后，Joplin 会定期自动同步。手机上打开 App，数据自动拉下来。

**导出/导入**：支持 JEX（Joplin 专有格式）、RAW（纯 Markdown 文件）、JSON。迁移到其他工具时可以导出纯 Markdown。

**笔记本 + 标签 + 待办**：三个维度管理笔记。笔记本管分类，标签管检索，待办管行动。互不依赖，灵活组合。

## 优缺点

**优点**：跨平台同步做得最扎实——手机电脑平板全覆盖，同步方式自主可控。开源免费，端到端加密是真正的隐私保护。Web Clipper 剪藏体验好，适合经常收集网上信息的用户。

**缺点**：界面比较朴素，不像 Obsidian 那样精致。没有双向链接和图谱视图，不适合构建知识网络。同步偶尔会冲突（两台设备同时修改同一篇笔记时）。插件生态不如 Obsidian 丰富。

## 小结

Joplin 最适合的场景：**需要在多个设备上随时记笔记、剪藏网页，又不想把数据交给任何一家云服务**。它不是知识管理工具，是笔记同步工具——用 Markdown 写，加密存，到处都能看。如果你对数据隐私敏感、设备跨平台，Joplin 很可能是你的主力笔记应用。
