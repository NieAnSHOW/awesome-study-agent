# Zettlr

## Zettlr 是什么

Zettlr 是面向学术写作的开源 Markdown 编辑器。不是 Obsidian 那样的知识管理工具，不是 MarkText 那样的极简阅读器——它的目标人群很明确：**写论文、做研究、管参考文献的人。**

Zettlr 会把你的论文写成 `.md` 文件，但导出时能变成 Word、LaTeX、PDF。你不需要学 LaTeX 语法，不需要手动排版参考文献，Zettlr 帮你干了这些脏活累活。

一句话说清楚：**用 Markdown 写论文，按一个键导出 Word，参考文献自动生成。**

## 核心特点

- **Zotero 文献管理集成**：这是 Zettlr 最大的卖点。装过 Zotero 的都知道，管参考文献是写论文最烦的事。Zettlr 直接连 Zotero 的数据库，你在编辑器里就能搜索、插入文献，不用切窗口
- **CiteKey 引用补全**：输入 `@` 符号，Zettlr 自动弹出引用建议。选中一条，引用格式自动生成。写 `[@smith2024]` 这样一串代码，导出时自动变成 `(Smith, 2024)` 这样的标准引用
- **导出 Word / LaTeX / PDF 等学术格式**：一个按钮，Markdown 变 Word 文档。内置 Pandoc 导出引擎，支持绝大多数学术期刊的格式要求。不是截图转文字那种"导出"，是真正的格式转换——标题、正文、引用、脚注，该有的都有
- **专注模式 + 黑暗模式**：专注模式只高亮当前正在写的那一句，其他行暗下去。不是花哨特效，是你写论文时确实需要的前面几段"只盯着眼前这一句"的状态
- **内置代码高亮**：如果你写的是计算机相关论文，代码块直接渲染高亮。省去拷到另一个工具截图再贴回来的步骤

## 快速上手

### 第一步：安装 Zettlr

从官网下载安装包，双击装好。打开后界面很干净——左边目录，中间编辑器，右边预览。跟其他 Markdown 编辑器差不多。

### 第二步：创建项目

点击"新建项目"，选一个文件夹作为你的论文目录。建议每个论文课题一个文件夹，Zettlr 会把它识别成一个项目。

### 第三步：写 Markdown

写论文正文就跟写笔记一样：`#` 表示标题，`**` 表示加粗。需要引用时，输入 `@` 搜索文献。Zettlr 的编辑器体验比 Word 好得多——没有格式工具栏干扰你，你只需要关注内容本身。

### 第四步：导出 Word

写完了？点一下导出按钮，选 Word 格式。Zettlr 自动生成带格式的 `.docx` 文件。交给导师、投期刊、打印，都行。

## 核心功能速览

**引用管理**：Zettlr 跟 Zotero 配合，等于给你的论文配了个专属图书馆管理员。搜索、插入、格式化，一气呵成。

**导出**：内置 Pandoc，支持 Word、LaTeX、PDF、HTML、Markdown 等多种格式。导出时自动处理引用、脚注、目录。

**专注模式**：只显示你正在写的段落。剩下内容淡出视线。适合写论文这种需要长时间专注的场景。

**黑暗模式**：长时间写论文不刺眼。晚上写论文的标配。

## 优缺点

**优点：**
- 写论文的体验远超 Word。纯文本写作不受格式干扰，你想的是内容，不是字号和行距
- 引用管理集成度行业最高。Zotero + Zettlr 的组合，能打 90% 的学术写作场景
- 导出质量好。不是凑合能看，是真的能交作业的格式
- 免费开源。不是"学生版免费写 14 天"，是真免费
- 跨平台。Windows、macOS、Linux 都能用

**缺点：**
- **第一次配置 Pandoc 需要一点耐心。** 导出依赖 Pandoc 引擎，安装和配置路径对新手不太友好
- 不是笔记管理工具。你想拿它当 Obsidian 用，会失望。没有图谱、没有插件生态、没有移动端
- 对中文支持可以，但中文学术引用格式需要手动调整
- 社区比 Obsidian 小很多。遇到问题，搜到的答案不如 Obsidian 多

## 一句话总结

Zettlr 不做"第二大脑"，它只做一件事——**让你用 Markdown 写论文，然后一键导出标准格式交给导师。** 如果你在写论文、做研究、被参考文献折磨，试试 Zettlr。不是替代 Word，是换一种更舒服的方式写论文。
