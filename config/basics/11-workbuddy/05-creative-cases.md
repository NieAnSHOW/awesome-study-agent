# 创意与设计案例

> **学习目标**: 掌握 WorkBuddy 在创意与设计场景中的实际应用
> **预计时间**: 30 分钟
> **难度等级**: ⭐⭐☆☆☆

本章收录 3 个创意与设计场景案例，涵盖海报生成、品牌文案创作和内容规划等任务。

---

## 案例 1：宣传海报生成

**一句话**：用 WorkBuddy 的设计能力直接生成宣传海报

**适用人群**：设计、运营

**前置条件**：明确的海报主题和文案

**指令示例**：

```
为 WorkBuddy 做一个产品宣传海报，风格：渐变玻璃卡片风格，画幅 3:4，内容突出"AI Agent 办公新范式"的主题。
```

**交付成果**：可直接使用的海报图片

---

## 案例 2：品牌文案创作

**一句话**：让 WorkBuddy 根据产品信息生成品牌宣传文案

**适用人群**：文案、市场

**前置条件**：产品/品牌基本信息

**指令示例**：

```
请写一篇 800 字左右的品牌故事，主题是"AI Agent 办公新范式"，介绍 WorkBuddy 如何改变职场人的工作方式，适合微信公众号发布。语言要亲切自然，有感染力。
```

**交付成果**：完整的品牌故事文章

---

## 案例 3：内容矩阵规划

**一句话**：让 WorkBuddy 帮助规划内容发布策略

**适用人群**：运营、新媒体

**前置条件**：明确的内容方向和目标

**指令示例**：

```
帮我规划一个月的内容发布计划，主题是关于"AI 办公效率提升"，包含每周的主题方向、内容形式（图文/视频/直播）、发布平台建议和预期目标。
```

**交付成果**：月度内容规划表

---

## 扩展思路

更多创意场景：
- **视频创作** → 短视频分镜脚本、视频标题优化、BGM 推荐

- **活动策划** → 活动方案大纲、宣传物料清单
