## 案例 1：行业调研报告

**一句话描述**: 让 WorkBuddy 自动搜索和分析行业信息，生成结构化的调研报告

**适用人群**: 市场、战略、产品经理

**前置条件**: 明确的调研主题

**核心能力**: 网页搜索 / 浏览器自动化 / NLP 分析 / 本地文件读写

### 场景说明

行业调研是市场、战略和产品经理的日常工作之一。传统做法是在多个信息源之间手动搜索、阅读、整理，一个完整的行业调研往往需要数小时甚至数天。借助 WorkBuddy，只需给出调研主题和关注方向，WorkBuddy 就能自动完成信息收集、数据整理和报告撰写。

## 案例 3：PPT 演示文稿制作

**一句话描述**: 基于调研内容让 WorkBuddy 自动生成精美的演示文稿

**适用人群**: 所有需要做汇报的职场人

**前置条件**: 已有调研内容或明确主题

**核心能力**: 文档处理 / API 调用 / 本地文件读写

汇报演示是职场中最常见也最耗时的任务之一。从零开始做一套精美 PPT，设计版式、提炼要点、配图排版，至少需要半天时间。WorkBuddy 可以基于已有的调研报告或直接根据主题，自动生成结构完整、风格统一、可直接使用的演示文稿。

## 扩展思路

以上 3 个场景可灵活组合和扩展。例如先执行调研分析，再基于结果生成 PPT，形成"调研→分析→汇报"的完整工作流。

更多商业场景：
- **Sales 场景** → 客户背景调研、销售话术准备
- **新媒体场景** → 爆款选题挖掘、多平台内容分发
- **品牌场景** → 品牌舆情监测、传播效果分析
